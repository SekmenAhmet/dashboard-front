# Dashboard Front

Frontend React + Vite + Tailwind pour le City Lifestyle Dashboard. Palette noire/gris/blanc, visualisations Plotly.

## Prérequis
- Node.js 18+

## Installation
```bash
cd dashboard-front
npm install
```

## Démarrer
```bash
# Backend attendu sur http://localhost:5000 (configurable)
npm run dev
```

## Configuration
- `VITE_API_BASE_URL` : URL de l'API backend (par défaut `http://localhost:5000`)

## Build
```bash
npm run build
npm run preview
```
